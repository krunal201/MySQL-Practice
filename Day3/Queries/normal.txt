Conclusion:
Tourist Hotel Search Project aims to give seamless experience for tourists who is looking for hotels near their destination.
Goal of this project is to make it easy for users to find hotels and complete bookings.

This project focus on user experience and make sure that the project is responsive across multiple devices.
Whenever a user searches for a hotels near their destination, they can see list of hotels with details like prices, ratings. 
Allowing users to make their travel budget friendly, find hotels and book hotels easily.

Tourist Hotel Search Project is made after understanding of user needs.
